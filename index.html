平台特点

从编写、运行到协作与统计，一站式覆盖测试用例全流程

用例与协作

多形态用例 · 灵活分享 · 多人协同

  • 脑图用例与 Excel 用例双模式
  • AI 一键生成用例，提升编写效率
  • 运行用例、开发自测与产品验收全支持
  • 全局 / 局部分享，局部用例与合并用例

日志、统计与导入导出

可追溯、可量化、多格式互通

  • 用例日志完整记录，变更可追溯
  • 用例统计看板，支撑决策与复盘
  • 多格式导入导出与另存为
  • 导出 JSON / PNG / SVG / PDF / Markdown / XMind / TXT，支持对应格式导入

脑图结构与节点

多结构、可定制、操作灵活

  • 逻辑结构图、思维导图、组织图、时间轴、鱼骨图等多种结构
  • 多主题内置，支持自定义样式与注册新主题
  • 节点支持富文本、图片、图标、链接、备注、标签、概要、公式

界面功能

多格式互通、高效操作与本地存储

  • 快捷键、撤销重做、关联线、搜索替换、小地图、水印、手绘与彩虹线
  • 工具栏：节点增删与编辑（图片、图标、链接、备注、标签、概要）
  • 侧边栏样式与结构面板、右键菜单、底部栏统计与视图切换；本地存储与文件读写
